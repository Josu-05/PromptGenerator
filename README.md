Este repositorio contiene el código de Prompt Generator, una herramienta que permite crear y organizar prompts de manera automática. El objetivo es facilitar la interacción con modelos de inteligencia artificial, generando instrucciones claras y adaptadas a distintos contextos.
En este ejemplo usamos un ejemplo de una concesionaria, en el cual se quiere facilitar la venta de vehículos.
Se usó StackLayout en el código.
